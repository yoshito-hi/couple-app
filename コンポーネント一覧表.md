# コンポーネント一覧表

Storybook ＋ CDD 開発で使う資料
作成したいコンポーネントが一覧で載っている

| カテゴリ              | コンポーネント名 | 役割・概要                                                                    | 基底とする部品  |
| :-------------------- | :--------------- | :---------------------------------------------------------------------------- | :-------------- |
| **Foundation (基底)** | `AppButton`      | ボタンの共通部品。テーマカラーの適用やホバー時の浮き上がり（2px）を制御する。 | shadcn `Button` |
|                       | `AppCard`        | 12-16pxの角丸と柔らかな影を持つ、アプリの核となるカード部品。                 | shadcn `Card`   |
|                       | `AppTypography`  | `H1`（Zen Maru Gothic）、`P`、`Muted`など、フォントと階層を定義する部品。     | -               |
|                       | `AppIcon`        | ログイン画面やエラー画面などで使用する、イラストやアイコンをラップする部品。  | -               |
| **Auth (認証)**       | `LoginButton`    | 「よしと」または「なほ」を選択するための専用ボタン。                          | `AppButton`     |
|                       | `AuthForm`       | メールアドレスとパスワードの入力を受け付けるフォーム部品。                    | shadcn `Form`   |
|                       | `AuthCard`       | ログイン画面中央に配置される、認証フォームを内包したカード。                  | `AppCard`       |
|                       | `AuthModal`      | 他の画面から未認証でアクセスした際に呼び出すダイアログ。                      | shadcn `Dialog` |
| **Timeline (年表)**   | `PeriodLine`     | 月ごとの区切りを表現するタイムラインの縦線。                                  | -               |
|                       | `MemoryCard`     | エピソードのタイトルやテキストを表示する、年表上の個別カード。                | `AppCard`       |
|                       | `InlineComic`    | 年表の中に埋め込まれる、漫画のサムネイルとタイトルのセット。                  | -               |
|                       | `ViewComicBtn`   | 漫画閲覧画面（SCR-003）へ遷移するための専用ボタン。                           | `AppButton`     |
| **Comic (漫画閲覧)**  | `PageCard`       | 漫画を読みやすく表示するための、詳細画面用メインカード。                      | `AppCard`       |
|                       | `ComicViewer`    | 4コマ漫画の画像を縦スクロールなどで閲覧可能にする専用ビューワー。             | -               |
| **Common (共通)**     | `GlobalHeader`   | アプリタイトル、戻るボタン、ログアウトボタンを含む共通ヘッダー。              | -               |
|                       | `ErrorState`     | エラー時に表示する、アイコンと再試行ボタンを含むカード部品。                  | `AppCard`       |
|                       | `NotFoundState`  | 404画面などで使用する、アイコンとホームへの導線を含むカード部品。             | `AppCard`       |

## ディレクトリ構成案

```markdown
src/
├── components/
│ ├── ui/ # 【Primitive】shadcn/ui の生の部品（基本触らない）
│ │ ├── button.tsx
│ │ ├── card.tsx
│ │ └── dialog.tsx
│ │
│ ├── base/ # 【Foundation】アプリ専用の基底部品
│ │ ├── AppButton.tsx # ui/button をラップ
│ │ ├── AppCard.tsx # ui/card をラップ
│ │ └── AppTypography.tsx
│ │
│ ├── features/ # 【Domain】特定の機能に紐づく部品
│ │ ├── auth/
│ │ │ ├── AuthForm.tsx
│ │ │ ├── AuthCard.tsx
│ │ │ └── LoginButton.tsx
│ │ ├── timeline/
│ │ │ ├── MemoryCard.tsx
│ │ │ └── PeriodLine.tsx
│ │ └── comic/
│ │ └── ComicViewer.tsx
│ │
│ └── layouts/ # 【Global】共通レイアウト部品
│ └── GlobalHeader.tsx
│
├── config/ # ユーザー設定やテーマの定数
└── hooks/ # カスタムフック（認証状態など）
```
